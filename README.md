# Daily Quest Tracker ⚡

A gamified productivity app with CBT support to help you build better habits through gameplay!

## 🎮 Features

- ✅ Daily goals with categories and difficulty levels
- ✅ XP & coin reward system
- ✅ Streak tracking for consistency
- ✅ 10 achievements to unlock
- ✅ Mood tracking with CBT support
- ✅ Low-energy "Minimum Victory" mode
- ✅ Todo/Grocery/Ideas lists
- ✅ 3 customizable themes
- ✅ Sound effects & animations
- ✅ Data export/import
- ✅ 100% offline - no data ever leaves your device

## 🚀 Quick Start

```bash
npm install
npm start
```

Opens at http://localhost:3000

## 📱 Use on Mobile

1. Get your computer's IP address
2. On mobile, visit `http://YOUR_IP:3000`
3. All data saves locally in browser

## 🌐 Deploy to Cloud

### Netlify (Easiest)
1. Push this repo to GitHub
2. Go to https://netlify.com
3. Click "New site from Git"
4. Select your repo
5. Deploy!

### Vercel
1. Push to GitHub
2. Go to https://vercel.com
3. Import project
4. Deploy!

## 🎯 How It Works

1. **Add daily goals** - Choose category & difficulty
2. **Complete goals** - Earn XP & coins
3. **Build streaks** - Consistency reward
4. **Unlock achievements** - Reach milestones
5. **Track mood** - Get CBT support when needed
6. **Level up** - Every 500 XP

## 🛠️ Tech Stack

- React 18
- Tailwind CSS
- Lucide React (icons)
- Local Storage (no backend needed)

## 📝 Data

All data is stored locally in your browser. No servers, no tracking, no data collection.

You can export your data as JSON anytime in Settings.
